## CLI Extract ISR Tables Tax Mexico

### Requirements

    - Python 3.6+
    - Pipenv

### Installation

    - Install Python 3.6+
    - Install Pipenv
    - Run `pipenv install`

### Usage

    - Run `pipenv shell`
    - Run `python isr-tables-extractor.py`

### Usage as CLI
    
    -  Dowload this repo
    -  Go to directory
    -  Run `pip install`
    -  Run `bin/isr-tables-extractor`
